# 🧩 MineMod

A simple Minecraft mod built with Java and Gradle.

## 📌 Overview
MineMod is a lightweight mod project for Minecraft, designed to experiment with custom features, gameplay mechanics, and Java modding structure using the Gradle build system.

---

## ⚙️ Features
- Custom mod structure using Forge/Fabric-style setup
- Gradle-based build system
- Easy to extend and modify
- Organized source code in `src/main`
